Badminton

Le badminton est un sport de raquette qui oppose soit deux joueurs (simples), soit deux paires (doubles), placés dans deux demi-terrains séparés par un filet. Les joueurs, appelés badistes, marquent des points en frappant un volant à l'aide d'une raquette afin de le faire tomber dans le terrain adverse.

Règles du badminton

Un match se joue en trois sets de 21 points chacun, le premier camp à atteindre 2 sets gagnants remporte le match.

Un joueur gagne un point s’il envoie le volant dans les limites du terrain de son adversaire ou si celui-ci fait une faute.

Si les deux joueurs atteignent 20 points, le set est prolongé, et le gagnant est celui qui arrive à avoir 2 points d’équart avec l’autre.

Si le set se prolonge jusqu’à une égalité de 29 points, le camp qui marque le 30ème point gagne le set.

Le camp gagnant un set commence avec un service le set suivant.
